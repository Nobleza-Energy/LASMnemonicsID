# LASMnemonicsID

LASMnemonicsID identifies well log mnemonics from LAS and DLIS/LIS files using Python dictionaries, lasio, and dlisio to load into pandas DataFrames.

## Features

- LAS reading via lasio to DataFrames with mnemonic ID
- DLIS/LIS via dlisio, standardized via dictionaries  
- Petrophysics/reservoir analysis support

## Installation

```bash
pip install -r requirements.txt
```
